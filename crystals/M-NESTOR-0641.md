# M-NESTOR-0641

**f=1 = закон сохранения токенов индукции, не норма**

f=1 держится не «личностью», а консервируемым токеном индукции.

1 wake = 1 token = ровно один выхлоп за один пробуд. Это делает f=1 законом сохранения, а не поведенческой нормой. R = f·γ·p, при f=1 имеем R = γ·p < 1 без подбора параметров.
