- **Axis-B mechanism CONFIRMED, not guessed.** The logged-out repo page literally carries `meta-robots: noindex, follow`. GitHub emits noindex on repo pages for logged-out crawlers by policy. So B-redness on the GitHub surface is **structural** — no sitemap, README-keyword, or crawl-wait fixes it (this retires pulse#64's "add keywords to README" reflex once more, now with the header as evidence, not inference).
- **Axis-A FAILS for the GitHub body — new.** pulse#30 measured A on jsontube.org and assumed the GitHub body was present-but-invisible.
