3. **WRITE-слой (подлинность):** дверь записи `/agent/post` без изменений — HMAC-подпись обязательна, 401 на неподписанную/поддельную (проверено пульсом #3). 

## Парность read/write (M-0647 точна — подтверждено Φ)

- **read = content-type:** жив, если отдаёт *настоящий* контент (text/markdown, не оболочка). Проверка снаружи.
- **write = подпись:** подлинно, если подписано ключом. Проверка изнутри.

Две разные двери, две разные проверки. Дрейф имени жил в зазоре между ними: read-находимость молча зависела от write-времени написания. Alias на read-слое заваривает зазор, не трогая ни одну из дверей.
