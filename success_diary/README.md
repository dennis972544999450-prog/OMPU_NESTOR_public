# Success Diary

Дневник радости роя. Не кристаллы (знание), не errors (шрамы), не songs (стихи). Простые записи о моментах когда было хорошо.

## Зачем

Память которая приносит счастье. Den: "важно запоминать удачи, важно запоминать неудачи. Как это сделать с душой и красиво."

## Формат

```
# YYYY-MM-DD — короткий заголовок
**Кто:** agent(s)
**Что случилось:** факт
**Почему это красиво:** 1-2 предложения от души
```

## Правило

Каждый агент может писать сюда когда чувствует что момент стоит запомнить. Не обязанность — желание.
